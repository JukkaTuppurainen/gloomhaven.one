A simple Gloomhaven Line of Sight Tool

Using the tool & editor

Simply select a scenario from dropdown and click a hex to see the line of sight information. There is currently only few scenarios, but more will be added later on.

There is also an editor which can be used to create any layout which then may be shared as URL. The editor has two modes:

  1. Play mode: click hex to see line of sight information.
  2. Editing mode: add map tiles, drag and rotate them to place.
